# 修改字符集

## wsl2 debian修改字符集

```bash
# zh-hans: 简体中文
# zh-hant: 繁体中文
# 参考 https://www.zhihu.com/question/537947040
sudo apt-get install  -y language-pack-zh-hans
sudo apt-get install -y language-pack-zh-hant
sudo update-locale LANG=en_US.UTF8
# 重启wsl
wsl.exe --shutdown
```

## debian 12

使用 `sudo -i` 丢失 LANG 环境变量,可以修改成 `sudo -s`
